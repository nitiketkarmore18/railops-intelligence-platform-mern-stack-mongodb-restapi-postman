# 🚆 RailOps Intelligence Platform

## Enterprise Rail Wagon Management & Predictive Analytics Platform

RailOps Intelligence Platform is a full-stack MERN application designed to modernize railway wagon operations through predictive maintenance intelligence, operational analytics, fleet health monitoring, and executive decision-support dashboards.

The platform enables railway administrators and maintenance teams to efficiently manage wagon records, monitor maintenance activities, track SLA compliance, forecast maintenance costs, analyze operational performance, and proactively identify high-risk assets before failures occur.

---

# 📌 Project Overview

Traditional wagon management systems often rely on manual monitoring and reactive maintenance strategies, leading to unexpected breakdowns, increased downtime, and operational inefficiencies.

RailOps addresses these challenges by integrating:

* Predictive Maintenance Intelligence
* Fleet Health Monitoring
* SLA Compliance Tracking
* Cost Forecasting
* Downtime Analytics
* Engineer Performance Analytics
* Executive KPI Dashboards
* Real-Time Operational Insights

The platform transforms raw operational data into actionable intelligence for maintenance planning and resource optimization.

# ⚙️ Installation Guide

### Clone Repository

```bash
git clone https://github.com/nitiketkarmore18/railops-intelligence-platform-mern-stack-mongodb-restapi-postman.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Configure Environment Variables

Create:

```bash
backend/.env
```

Add:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
